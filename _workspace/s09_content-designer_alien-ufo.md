# 외계인 UFO 탑승 외형 — 3샘플 비교 (검토용, 미확정)

외계인(alien)을 **UFO(비행접시) 탑승** 외형으로. 외형만 변경 — 스킬(mimic)/엔진/밸런스/다른 캐릭터 불변.
`src/data/partmodels/alien.ts` + `src/data/characters/alien.ts` 팔레트만 수정. glide runStyle 유지.

각 샘플은 같은 rig(파츠 이름 동일)로 구현 → 렌더러의 glide 프로시저(다리 tuck=접시 하단 라이트, 안테나 sway, body breathe/tilt=접시 흔들림)를 그대로 활용. 렌더러 무수정.

## 육안 확인 결과 (Read로 직접 확인)
- **1**: 접시 확실히 읽힘. 은빛 디스크 + 림 포트홀 + 돔 안 외계인. 돔 투명도가 낮아 약간 은은(초록 머리·빨간 트랙 위라). 파츠 깨짐 없음.
- **2**: 외계인 전신이 접시 위에 앉은 큐트 토이 느낌. 라벤더가 초록 외계인과 대비 좋음. 접시가 1번보다 작아 "위에 탑승" 톤. 깨짐 없음.
- **3**: 네온 시안 림 + 언더글로우 헤일로로 가장 "쿨/테키". 다크 헐과 발광 대비 선명. 게임 스케일 보드샷에서도 접시·발광 또렷, 깨짐 없음.
- 셋 다 typecheck 통과, glide 떠다님/안테나 sway 정상, 결정론 무영향.

## 현재 적용 상태
- 소스에는 **샘플 3 (네온 글로우 림)** 가 적용되어 있음. **확정 아님** — 최종 선택은 사용자.
- 다른 샘플로 바꾸려면: partmodels/alien.ts를 해당 샘플 블록으로, characters/alien.ts 팔레트의 hull/hullTop/dome(/rim) 4줄을 해당 샘플 값으로 교체.
