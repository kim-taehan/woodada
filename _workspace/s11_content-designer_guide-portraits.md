# s11 content-designer — 가이드 포트레이트 (거미 신규 + 고양이 갱신 + alien 신규 + eagle 정리)

## 작업 결과
`docs/character-guide.md`가 참조하는 포트레이트 중 깨지거나 낡은 것만 서지컬하게 처리.

| 파일 | 처리 | 내용 |
|------|------|------|
| `docs/img/spider.png` | **신규 생성** | 현재 거미(밝은 파스텔 라벤더 #9B7FD4, abduct) 등장 프레임 크롭. 230×200. |
| `docs/img/cat.png` | **갱신(교체)** | 옛 회색 고양이 → 현재 검은 고양이(노란눈, catwalk)로 재캡처. 230×200. |
| `docs/img/eagle.png` | **삭제** | 고아 에셋. 가이드 본문에 `eagle`/`독수리` 참조 없음을 `grep`으로 확인 후 제거. |
| `docs/img/alien.png` | **신규 생성(후속)** | UFO 샘플3 확정(다크 틸 헐 #244A52 + 네온 시안 림 #3DF2E0 + 언더글로우) 외계인 캡처. 230×200. |

손대지 않은 것: `dog.png`, `monkey.png`, `bear.png`, `penguin.png`, `hedgehog.png` (타임스탬프·크기 그대로). 이로써 가이드의 마지막 깨진 이미지(alien)까지 모두 채워짐.

## 캡처 방식
- 임시 spec `tests/e2e/guide-portrait-tmp.spec.ts`(작업 후 삭제)로 `window.__woodada` 훅 사용.
- 로스터 `['cat','spider']`만 띄워 두 캐릭터가 크고 또렷하게 잡히도록 함. seed 3.
- 풀보드를 0.36~0.56 구간 2%씩 스윕 → **seed 3, 52% 프레임**에서 둘 다 상단 직선주로에 스킬 FX 없이 깔끔하게 위치(거미 위·고양이 아래로 세로 분리)인 것을 선택.
- Playwright `clip`으로 정확히 230×200 영역을 직접 잘라 저장(sips 후처리 불필요).
  - 거미: clip `{x:573, y:90, w:230, h:200}` — "거미2" 이름표 + 라벤더 치비 중앙.
  - 고양이: clip `{x:700, y:150, w:230, h:200}` — "고양이1" 이름표 + 검은 고양이 갤럽 포즈 중앙.
- **외계인(후속)**: 로스터 `['alien','monkey']`, seed 5. 외계인이 mimic을 자주 써 스킬 ✨가 잘 끼므로 49~60% 1%씩 미세 스윕 → **57% 프레임**(외계인 상단 직선주로, 스킬 발광 없음, 접시·네온 림·돔·안테나 또렷)에서 clip `{x:428, y:128, w:230, h:200}`.
- 임시 spec과 스크래치 디렉터리(`_workspace/portrait/`) 모두 삭제 완료.

## 육안 확인 (Read로 직접 열어봄)
- **거미**: 라벤더 치비가 "거미2" 이름표 아래 중앙. 붉은 트랙·레인 스트라이프 배경이 기존 dog/hedgehog 포트레이트와 톤·프레이밍 일치. 우하단 모서리에 옆 고양이 이름표("고") 작은 조각이 살짝 걸치나 가장자리라 미미.
- **고양이**: 검은 몸·노란눈 고양이가 "고양이1" 이름표 아래 중앙, 갤럽 포즈 또렷. 좌상단에 거미 다리 작은 조각, 하단에 인필드 초록 약간 — 둘 다 가장자리로 미미(기존 dog 포트레이트도 아이템박스/레인 요소가 함께 보이는 캐주얼 캡처 톤).
- **외계인(후속)**: 초록 외계인(안테나+노란 전구) 탑승, 다크 틸 UFO 헐 + 네온 시안 발광 림/언더글로우 + 돔 유리가 또렷이 읽힘. "외계인1" 이름표 아래 중앙, 붉은 트랙 배경으로 기존 톤·프레이밍 일치. 좌상단 모서리에 옆 원숭이 작은 조각만 살짝(거미/고양이와 동급 가장자리 미미).
- 셋 다 `sips`로 230×200 재확인. `docs/img/`에 설치 후 다시 Read로 정상 렌더 확인.
