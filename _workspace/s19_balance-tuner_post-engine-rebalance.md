# s19 · 엔진 묶음 후 전반 재밸런스 (balance-tuner)

엔진 묶음(캣워크 반응형·쿨다운 필드스케일·스킬후 0.3s 무적·바나나 양방향·스턴 시 쿨다운 리셋) + 외계인 너프 적용된 최종 엔진 기준 재측정·조율.

## 진단 (baseline)
- 로스터 8종: dog cat monkey bear penguin hedgehog spider alien (penguin 이미 반영됨).
- **유일한 독주: bear.** 팀 랭크합산(동질 2쌍) win share **2.97× / 3.64× / 3.60×** (laps 1/3/10). 개인전 1.96/1.79/1.84×, 릴레이 1.94/1.63/1.22×.
  - 원인: 포효가 범위 내 전원을 staggerMs 동안 스턴 → 밀집한 동질팀 필드에서 두 곰의 포효가 팩 전체를 마비, 곰은 power5+저속순항으로 면역. 밀집 필드에서 과강.
- penguin: 모드 전반 floor 근처(0.46~0.84×)지만 하니스 floor(0.4× fair) 아래로는 안 떨어짐 — 경계선, 우선순위 아님.
- 바나나 양방향/캣워크/무적/쿨다운리셋發 신규 쏠림: **없음.** monkey·cat·spider 모두 정상 범위.

## 조정 (bear.ts skill.params·cooldownMs만)

|---|---|---|---|
| before | [3000, 5200] | 28 | 430 |

다른 캐릭터·엔진·핸들러·테스트 무수정. 외계인 너프(speed2·쿨다운↑) 유지.

## 결과 (win share, fair×)
| 모드·laps | bear before | bear after |
|---|---|---|
| 개인 1 | 1.96× | **1.15×** |
| 개인 3 | 1.79× | **1.14×** |
| 개인 10 | 1.84× | **1.14×** |
| 팀 1 (N=3000) | 2.97× | **1.61×** |
| 팀 3 (N=1500) | 3.64× | **2.05×** |
| 팀 10 (N=1500) | 3.60× | **2.23×** |
| 릴레이 1 | 1.94× | **1.15×** |
| 릴레이 3 | 1.63× | **1.04×** |
| 릴레이 10 | 1.22× | **0.88×** |

- 명시 이슈(곰 팀 동질모드 천장 ~2.0× 누르기): baseline은 실제 **3.6×**였고 → **1.6×(1lap)·2.05~2.23×(3/10lap)**로 대폭 완화.
- 개인전·릴레이는 모두 fair(0.88~1.15×)로 정상화. 곰을 약체로 떨어뜨리지 않음(균형 유지).
- 잔여 팀 3/10lap의 ~2.0~2.2×는 **동질 곰2쌍이 밀집팩에 포효를 스택하는 적대적 워스트케이스**(N=1500 안정 측정). 혼합 팀에선 곰 독주 없음. "특정 조합이 강한 건 허용, 누구도 일방 독주 안 함" 철학상 허용 범위 — 더 누르면 곰이 정작 개인/릴레이에서 약체가 됨.

## 게이트
- `engine-bias.test.ts` **7/7 통과**.
- `npm run typecheck` 통과.
- penguin-stack 회귀: A_pengStack 15% / B_mixed 23% / C_mixed 62% — 펭귄 스택 독주 없음(정상).

## 사용자 결정: wide 복원 (2026-06-22)
사용자가 곰 포효를 **넓게 복원**(잘 걸리게) + **동질 곰2:2 팀 워스트케이스 강함 수용**(실전 거의 없음)으로 결정. bear.ts roar를 task #12 wide 값으로 되돌림: `range 22→28`, `staggerMs 300→430`, `cooldownMs [3800,6200]→[3000,5200]`. range 28에서도 engine-bias(개인전) **7/7 통과**·typecheck 통과 확인 → 그대로 확정(최소 조정 불필요). 즉 위 rebalance 표는 측정 기록으로 남기되, **현재 적용값은 wide 복원본**임.
