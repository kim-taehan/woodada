# s08 — character-guide.md 능력치 표기 정리

대상: `docs/character-guide.md` (문서 전용, 코드/데이터/엔진 무관)

## 1. 속도 표기 단일화 (중복 제거)

- **결정**: `🏃 속도 / 🛡️ 파워` 체계로 단일화. **`⚡스피드 💥파워 🎲변수 🛡️생존` 4축 A~D 표를 전면 삭제**.
- **근거**:
  - "스피드(4축)"와 "속도(스탯 줄)"가 같은 stat을 두 이름으로 부르던 중복의 본체가 4축 표였음.
  - 4축 표의 `💥파워`/`🛡️생존` 역시 스탯 줄의 `파워`와 의미가 겹쳐 혼란 가중.
  - 속도/파워 2축은 schema의 `speed`/`power`와 1:1 직결 → 데이터와 일관, 팀 리더 권장안과 일치.
- 결과적으로 캐릭터별 4축 표 블록 7개 전부 제거, 헤더 안내문도 4축 설명(`⚡스피드 · 💥파워 · 🎲변수 · 🛡️생존`) 줄 삭제하고 속도/파워 설명만 남김.

## 2. 수치 제거 (밸런스 보류)

- `●●●●● 5`, `●○○○○ 1` 같은 **불릿/숫자 등급 전부 제거**, A~D 등급도 4축 표와 함께 제거.
- **자리(섹션 골격)는 유지**: 각 캐릭터에 `🏃 속도 / 🛡️ 파워` 줄은 남기되 정성 설명 + `*(수치는 추후 밸런스 확정 시 기입)*` placeholder로 대체.
- 헤더 안내에도 `*구체 수치는 밸런스 확정 시 일괄 기입 예정이에요.*` 한 줄 추가.
- 한눈에 보기 표에서도 `🏃 속도 / 🛡️ 파워` 수치 열 삭제(스킬 열까지만).
- **참고**: 기존 문서의 곰 수치(속도1/파워5)가 현재 데이터(speed2/power5)와 불일치했는데, 수치를 전부 빼면서 자연히 해소됨. 정성 표현은 "느리지만 끄떡없는 탱크"로 유지(기존 "제일 느리지만" → "느리지만"으로만 완화).

## 3. 로스터 동기화

- **독수리(eagle) 삭제**: 한눈에 보기 표 행 + `## 🦅 독수리` 섹션 통째 제거.
- **거미(spider)·외계인(alien) 추가**: 데이터(`src/data/characters/spider.ts`, `alien.ts`) 기준으로 한 줄 소개·스킬 설명·tip 신규 작성.
  - 🕷️ 거미 — 거미줄(위치 강등·속박): 선두를 자기 등 뒤로 끌어내리고 묶음. 역할=단일표적 위치 강등.
  - 👽 외계인 — 의태(스킬 카피): 가장 가까운 동물 스킬을 베껴 발동하는 변수형 와일드카드.
- 최종 로스터 8종 일치: 강아지·고양이·원숭이·곰·펭귄·고슴도치·거미·외계인.

## 검증

- 마크다운 표(한눈에 보기 4열) 정렬·헤더 깨짐 없음 확인.
- 문서라 typecheck/test 무관.
- 이모지 스타일·톤·섹션 구조(`## 이모지 이름` + `![](img/..)` + 스킬/스탯/tip) 기존 그대로 유지, 외과수술식 변경.
