# 우다다 — 동물 & 스킬 카탈로그 (20종)

> 작성일 2026-06-17 · 귀여움 × 병맛 코믹 톤 · 데이터 주도(데이터 + 스킬 핸들러)로 확장.

스킬은 **눈에 확 읽히게**(과장된 비주얼 + 말풍선 + 위치/속도 변화)가 원칙입니다(스펙 §2.3).
캐릭터당 스킬 후보 3-4개 중 ⭐는 추천 1순위(현재 로스터 역할 균형 기준).

## 역할 태그 범례
| 태그 | 뜻 |
|---|---|
| `부스트` | 자기 가속 |
| `변칙` | 랜덤·자기발목·순간이동 등 변수 |
| `방해·저격` | 한 명을 원거리로 멈춤/방해 |
| `방해·광역` | 주변 여러 명을 동시에 방해 |
| `방해·근접` | 붙은 동물을 밀치기/끌기 |
| `방어` | 방해 면역 / 안 막힘 (탱크) |
| `지원` | 남을 도움(이타) |
| `함정` | 트랙에 함정 설치 |

## Part 1. 기존 5종 (스킬 점검 + 대안)

### 🐶 강아지 — *우다다(Zoomies)* · 게임 마스코트
산만함의 화신. 이유 없이 폭주.
- ⭐ **우다다** `부스트·변칙` ✅구현 — 갑자기 통제불능 폭주, 가끔 옆길로 샘.
- **물어오기** `변칙` — 가까운 아이템 박스로 자동으로 끌려가 수집.
- **꼬리 돌풍** `방해·근접` — 뒤 동물 시야를 꼬리로 가림.
- *점검: 마스코트 정체성이라 현행 유지 권장.*

### 🐰 토끼 — *방심 낮잠(Nap)*
자만하는 허당. 졸다 깨다.
- ⭐ **방심 낮잠** `변칙` ✅구현 — 선두권일 때 졸다가 화들짝 깨서 대시.
- **깡총 연속 점프** `부스트` — 짧게 통통 튀며 가속.
- **당근 와작** `방어` — 당근 먹고 잠깐 방해 면역 + 기력 회복.
- *점검: 토끼와 거북이 우화 정체성 → 현행 유지 권장.*

### 🐒 원숭이 — *바나나 던지기(Banana)*
장난기 폭발 트러블메이커.
- ⭐ **바나나 던지기** `방해·저격` ✅구현 — 앞/뒤 표적 스턴(회피 가능, 헛던짐 개그).
- **나무 타기** `변칙` — 잠깐 공중으로 떠 레인 무시 추월.
- **흉내내기** `변칙` — 옆 동물의 스킬을 따라 씀.
- *점검: 저격 방해 유일 → 현행 유지 권장.*

### 🐘 코끼리 — *철벽(Brace)*
든든한 탱크.
- ⭐ **철벽 모드** `방어` ✅구현 — 잠깐 무적, 방해 면역 + 안 막힘, 앞을 밀고 감.
- **코 분수** `방해·광역` — 물 뿌려 주변 시야 가림.
- **쿵 발구르기** `방해·근접` — 지진으로 주변 비틀.
- *점검: 방어 역할 유일 → 현행 유지 권장(겹침 방지).*

### 🐻 곰 — *포효(Roar)*
묵직한 산의 왕.
- ⭐ **포효** `방해·광역` ✅구현 — 주변 동물 다 같이 움찔(짧은 스턴).
- **꿀 충전** `부스트` — 꿀 먹고 든든하게 가속.
- **곰손 후려치기** `방해·근접` — 옆 동물 밀치기.
- *점검: 광역 방해 정체성 → 현행 유지 권장.*

## Part 2. 신규 15종

### 🦔 고슴도치 (Hedgehog)
"붙지 마. 진짜."
- ⭐ **가시 밀치기** `방해·근접` — 가까이 붙은 동물을 가시로 옆/뒤로 밀어냄.
- **웅크리기** `방어` — 공처럼 말려 방해 면역 + 데굴 가속.
- **가시 발사** `방해·저격` — 뒤로 가시 한 발 쏴 추격자 멈춤.

### 🦅 독수리 (Eagle)
"내 거다. (콱) …어, 돌이네."
- ⭐ **낚아채기** `방해·변칙` — 앞서가는 동물 하나를 낚아채 뒤에 떨굼.
- **급강하** `부스트` — 하늘로 떴다 내리꽂으며 가속.
- **매서운 눈빛** `방해·저격` — 한 명 겁줘 잠깐 멈춤.

### 🐆 치타 (Cheetah)
"세상에서 제일 빠름. 잠깐만."
- ⭐ **초가속 스프린트** `부스트` — 폭발적으로 짧고 강하게 가속.
- **지친 숨** `변칙` — 스프린트 후 헥헥대며 잠깐 느려짐(자기발목).
- **사냥 집중** `부스트` — 앞 동물에 가까울수록 더 빨라짐.

### 🐢 거북이 (Turtle)
느리지만 우직하게.
- ⭐ **등껍질 숨기** `방어` — 등껍질에 쏙 들어가 방해 면역.
- **우직한 뒷심** `변칙` — 평소 느리지만 막판에 꾸준히 역전.
- **등껍질 회전** `방해·근접` — 옆으로 굴러 부딪혀 밀어냄.

### 🐧 펭귄 (Penguin)
"직진은 자신 있어!"
- ⭐ **배 썰매** `부스트·변칙` — 배로 미끄러져 직선 폭주(곡선에선 약함).
- **얼음 함정** `함정` — 얼음판 깔아 뒤 동물 미끄러뜨림.
- **무리 본능** `지원` — 가까운 펭귄끼리 같이 가속.

### 🐿️ 다람쥐 (Squirrel)
볼주머니 빵빵.
- ⭐ **도토리 폭격** `방해·저격` — 도토리 여러 발 난사로 앞 동물들 견제.
- **볼주머니 대시** `부스트` — 볼 가득 채우고 단거리 폭주.
- **나무 숨기** `변칙` — 잠깐 사라졌다 앞에서 등장.

### 🐍 뱀 (Snake)
스르륵.
- ⭐ **칭칭 감기** `방해·근접` — 앞 동물을 휘감아 잠깐 정지.
- **독니** `방해·저격` — 물어서 지속적으로 느리게.
- **또아리 부스트** `부스트` — 똬리 틀었다 튕기듯 가속.

### 🦎 카멜레온 (Chameleon)
지금 어딨게?
- ⭐ **투명 위장** `변칙·방어` — 잠깐 투명해져 방해 표적에서 제외 + 슬쩍 전진.
- **긴 혀 낚시** `방해·저격` — 혀로 앞 동물을 끌어와 자리 교환.
- **색 변신** `지원` — 주변 색에 섞여 들키지 않음(개그 연출).

### 🦘 캥거루 (Kangaroo)
콩! 콩! 콩!
- ⭐ **점프 도약** `부스트·변칙` — 크게 점프해 한 구간을 건너뜀.
- **주머니 태우기** `지원` — 꼴찌 동물을 주머니에 태워 같이 끌어줌.
- **뒷발 차기** `방해·근접` — 뒤 동물을 뻥 차서 밀어냄.

### 🐙 문어 (Octopus)
먹물 한 방.
- ⭐ **먹물 뿌리기** `방해·광역` — 주변 화면을 먹물로 가려 혼란.
- **다리 8개 질주** `부스트` — 다리로 촤르륵 미끄러져 가속.
- **빨판 붙잡기** `방해·근접` — 앞 동물을 잡아 끌어당김.

### 🦨 스컹크 (Skunk)
가까이 오지 마…
- ⭐ **방귀 가스** `방해·광역` — 뒤로 가스 뿜어 추격자들 느려짐.
- **냄새 지뢰** `함정` — 구간에 악취 함정 설치.
- **줄행랑** `부스트` — 방귀 반동으로 본인 가속(개그).

### 🦊 여우 (Fox)
속았지?
- ⭐ **속임수 페이크** `변칙` — 가짜 분신을 만들어 표적 교란(방해 회피).
- **잽싼 새치기** `부스트` — 빈틈으로 쏙 끼어들어 추월.
- **꼬리 홀리기** `방해·저격` — 한 명 홀려 잠깐 멈춤.

### 🦥 나무늘보 (Sloth)
…천천히 갈게…
- ⭐ **슬로우 → 각성** `변칙` — 초반엔 굼뜨다 막판에 갑자기 각성 폭주(대반전).
- **느긋함** `방어` — 너무 느긋해서 방해가 안 통함(개그 면역).
- **매달리기** `변칙` — 앞 동물에 매달려 무임승차.

### 🐎 말 (Horse)
정통 레이서.
- ⭐ **2단 가속 질주** `부스트` — 꾸준한 고속 + 한 번 더 치고 나감.
- **뒷발 차기** `방해·근접` — 뒤 동물을 뻥.
- **선두 본능** `변칙` — 뒤처질수록 빨라짐(고무줄 역전).

### 🦫 두더지 (Mole)
어, 없네? (퐁)
- ⭐ **땅굴 순간이동** `변칙` — 땅을 파고 들어가 앞쪽에서 솟아남.
- **흙 뿌리기** `방해·저격` — 뒤로 흙 뿌려 추격자 시야 방해.
- **갑툭튀** `방해·근접` — 갑자기 솟아 옆 동물 놀래켜 비틀.

### 🦦 수달 (Otter)
미끄러지는 게 제일 빨라.
- ⭐ **미끄럼 활강** `부스트` — 배로 쭉 미끄러지며 가속.
- **조개 던지기** `방해·저격` — 배에 얹은 조개를 던져 견제.
- **물장구 응원** `지원` — 가까운 동물 살짝 밀어줌(이타).

## 역할 분포 요약 (⭐추천 기준, 20종)

| 역할 | 캐릭터 |
|---|---|
| 부스트 | 강아지·치타·말·수달 (+펭귄/캥거루 겸) |
| 변칙 | 토끼·여우·나무늘보·두더지 (+강아지/카멜레온 겸) |
| 방해·저격 | 원숭이·다람쥐 |
| 방해·광역 | 곰·문어·스컹크 |
| 방해·근접 | 고슴도치·뱀 |
| 방해·변칙 | 독수리 |
| 방어 | 코끼리·거북이·카멜레온 |
| 지원 | (캥거루·수달 보조 옵션) |

> 방해가 다소 많고 **지원/함정**이 얇음 → 차별화를 원하면 펭귄(얼음 함정)·캥거루(주머니 태우기)·수달(응원)의 보조 옵션을 살리면 새 결이 생김.

## 구현 메모 (스킬 핸들러 재사용성)
- **기존 핸들러 재사용으로 즉시 추가 가능**(데이터만):
  - `banana`(단일 스턴) → 다람쥐 도토리, 독수리 눈빛, 여우 꼬리, 수달 조개
  - `roar`(광역 스턴) → 스컹크 방귀, 문어 먹물, 코끼리 코분수
  - `brace`(방어) → 거북이 등껍질, 고슴도치 웅크리기, 나무늘보 느긋함
  - `zoomies`(부스트) → 치타, 말, 펭귄 배썰매, 수달 활강
- **새 핸들러 1개씩 필요**(스펙 §2.4: 데이터 + 핸들러):
  - `snatch`(낚아채 뒤로 떨굼) — 독수리 ⭐
  - `shove`(근접 밀치기) — 고슴도치 ⭐ / 뱀 / 캥거루 뒷발
  - `teleport`(순간이동) — 두더지 ⭐ / 다람쥐 나무숨기
  - `vanish`(투명·표적제외) — 카멜레온 ⭐
  - `trap`(함정 설치) — 펭귄 얼음 / 스컹크 지뢰
  - `draft`(슬립스트림·이타) — 캥거루 주머니 / 수달 응원 / 펭귄 무리
  - `lastSpurt`(고무줄 역전) — 말 / 나무늘보 / 거북이 뒷심

> 권장 추가 순서: ① 기존 핸들러 재사용 5~6종 먼저(빠른 다양성) → ② `snatch`·`shove`·`teleport` 새 핸들러 → ③ `trap`·`vanish`·`draft`·`lastSpurt`(연출·UX 더 필요).
