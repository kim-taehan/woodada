# 우다다 캐릭터 로스터 (Character Roster)

귀여운 동물들이 원형 트랙을 달려 순위를 정하는 **동물 경주 추첨 게임**. 이 문서는 현재 6종 로스터의 외형·스킬·대사·능력치를 데이터에서 직접 읽어 정리한 단일 레퍼런스다.

> 데이터 출처: `src/data/characters/<id>.ts`, `src/data/partmodels/<id>.ts`, `src/engine/skills/<type>.ts`

## 능력치 모델 (중요)

이 게임에는 **캐릭터별 고정 기본 능력치가 없다.** 모든 레이서의 기본 순항 속도(`baseSpeed`)는 엔진(`RaceEngine.ts`)에서 `[1.3, 1.5]` 범위의 **균등 지터**로 매 경주마다 무작위 배정되며, 캐릭터 ID와 무관하게 동일한 밴드를 쓴다(인코스 쏠림·편향 방지). 프레임마다 추가 `SPEED_JITTER`도 적용된다.

따라서 캐릭터를 **차별화하는 것은 세 가지**다:

1. **`renderScale`** — 화면상 크기(시각적 존재감). 속도엔 영향 없음.
2. **스킬(`skill.type` + `params`)** — 부스트/방해/회피/지역효과 등 실제 게임플레이 차이의 핵심.
3. **쿨다운(`cooldownMs`)** — 스킬을 얼마나 자주 쓸 수 있는지. `[min, max]` 범위에서 매번 무작위.

> 밸런스는 보류 중: 정밀 승률 튜닝 대신 느슨한 공정성만 유지하며, 스킬 수치는 모두 `params`로 빼서 추후 튜닝한다.

- **runStyle**: `gallop` (측면 4족 갤럽 — 렌더러가 진행 방향에 맞춰 좌우 반전).
- **partModel**: `dog` — 측면 치비. 큰 둥근 머리 + 큰 반짝이는 눈 + 블러시 + 작은 주둥이, 통통한 몸과 짧고 뭉툭한 다리. 앞다리 쌍(`frontLegL`/`frontLegR`) + 뒷다리 쌍(`legL`/`legR`)을 갤럽으로 스윙. 먼 쪽 다리는 살짝 어둡게(`#E6C896`) 깊이감. 꼬리 한 줄.

- 한 줄 외형: 혀를 내민 연베이지 강아지, 측면으로 신나게 달리는 모습.

### 스킬 — `zoomies` (부스트·변칙)
- 갑작스러운 통제 불가 폭주: `[burstMin, burstMax]` 범위의 추가 전진 속도를 `burstMs` 동안 부여(`phase='straying'`). `strayChance` 확률로 레인이 `strayLane`만큼 바깥(아웃코스)으로 휙 샌다. 크고 잘 보이는 연출.
- **cooldownMs**: `[3000, 6000]`

| param | 값 | 의미 |

| `burstMax` | 1.1 | 추가 전진 속도 최대 |
| `burstMs` | 520 | 폭주 지속 시간(ms) |
| `strayChance` | 0.32 | 바깥으로 샐 확률 |
| `strayLane` | 0.45 | 샐 때 레인 이동량(0=인코스~1=아웃코스) |

| `dodgeChance` | 0.6 | 들어오는 방해를 회피할 확률 |
| `slipBoost` | 0.16 | 윈도우 동안 작은 전진 부스트(units/frame) |

## 🐧 펭귄 (penguin)

### 이미지 / 렌더링
- **runStyle**: `biped` (정면 뒤뚱이 — 렌더러가 다리 교대 + 플리퍼 스윙).
- **partModel**: `penguin` — 정면 치비. 둥근 검은 등 몸통에 큰 크림 배, 검은 캡이 흰 얼굴을 감싼 둥근 머리, 큰 눈, 작은 주황 삼각 부리. 짧은 플리퍼 두 개(`armL`/`armR`)가 뒤뚱 스윙, 뭉툭한 주황 물갈퀴 발(`legL`/`legR`), 짧은 꼬리. 독수리(펼친 날개·공중·브로우)와 달리 서서 뒤뚱, 배가 지배적 크림 실루엣.
- **renderScale**: `0.9`.
- **palette**: base `#2B3540`(검은 등/캡), point `#F4F1EA`(흰 배/얼굴), outline `#161C24`, cheek `#F2A6B6`, beak `#F2972F`(주황 부리+발), water `#5BC8E8`(빙판 존 틴트 — 렌더러 FX용).
- 한 줄 외형: 검은 등에 큰 흰 배, 주황 부리·발의 뒤뚱이 펭귄, 정면으로 종종걸음.

### 스킬 — `icefield` (빙판 지역 / 종족 감속)
- 자기 **앞쪽**에 미끄러운 빙판을 깖(시작 = `progress + aheadOffset`, 길이 `zoneLength`, `durationMs` 동안). **팀이 아닌 종족 기준**: `characterId === 'penguin'` 인 펭귄은 가속(`boostFactor` — 홈 빙판 활강), 그 외 전원은 감속(`slowFactor` — 미끄러짐). 순수 환경 효과(팀 무관, 솔로/팀/릴레이 동일)이며 **직접 방해가 아니라 캣워크 회피가 적용되지 않음**. 위치는 진행도 기반 완전 결정론(RNG 없음).
- **cooldownMs**: `[5000, 8000]` (로스터 최장)

| param | 값 | 의미 |
|---|---|---|
| `zoneLength` | 130 | 빙판 존 길이 |
| `durationMs` | 3200 | 빙판 유지 시간(ms) |
| `slowFactor` | 0.85 | 비펭귄 속도 배율(감속) |
| `boostFactor` | 1.05 | 펭귄 속도 배율(가속) |
| `aheadOffset` | 40 | 펭귄 앞쪽 빙판 시작 오프셋 |

### 대사 (`lines`)
- skill: `빙판 깔기! 미끌미끌~ 🧊`
- win: `뒤뚱뒤뚱, 1등 펭귄!`
- lose: `미끄러졌다… 삐악…`
- dodge: `얼음은 내 홈이지롱~`
