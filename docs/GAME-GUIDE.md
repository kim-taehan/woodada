# 🐾 우다다 (Woodada) — 완전 가이드

귀여운 동물들이 원형 트랙을 달려 순위를 정하는 **동물 경주 추첨 게임**.

🎮 **바로 플레이하기**: [https://kim-taehan.github.io/woodada/](https://kim-taehan.github.io/woodada/)

---

## 🎮 게임 방법

1. **참가자 이름 입력** (2~8 명)
2. **출발 버튼 클릭** (캐릭터는 랜덤 자동 배정)
3. **3·2·1 카운트다운** → 경주 시작!
4. **결과 확인** — 1 등부터 꼴찌까지 순위 공개

> 💡 **3 초 컷 원칙**: 앱 열자마자 3 초 안에 출발 가능. 모든 설정은 기본값으로 자동 처리됩니다.

---

## 🏆 게임 모드

### 개인전
모든 참가자가 개별 순위로 겨룹니다.

### 팀전
참가자를 팀으로 묶어 팀 단위 승부를 봅니다.
- **1 등 보유**: 각 팀에서 가장 빠른 동물의 등수로 비교
- **등수 합**: 팀원 등수 합이 가장 작은 팀 우승
- **이어달리기**: 릴레이 방식 (확장 예정)

## 🐾 캐릭터 가이드

총 **8 마리**의 동물, 각자 고유 스킬로 순위가 뒤집힙니다!

> **🏃 속도 / 🛡️ 파워** — 속도는 초반 페이스, 파워는 몸싸움 저항 (밀치기·스턴 덜 휘둘림). 누구도 일방적으로 유리하지 않습니다.

| 동물 | 스킬 | 한 줄 소개 |
|---|---|---|
| 🐶 **강아지** | 우다다 | 이유 없이 폭주하는 산만함의 화신 |
| 🐱 **고양이** | 캣워크 | 방해 따윈 확률로 피하는 얄미운 추월자 |
| 🐒 **원숭이** | 바나나 | 콕 집어 미끄러뜨리는 장난꾸러기 |
| 🐻 **곰** | 포효 | 포효 한 방에 주변 다 같이 움찔 |
| 🐧 **펭귄** | 빙판 | 앞에 빙판을 깔아 추격자를 늪에 빠뜨림 |
| 🦔 **고슴도치** | 가시 | 뒤에서 추격하면 가시로 톡 밀어냄 |
| 🕷️ **거미** | 거미줄 | 선두를 콱 낚아채 뒤로 끌어내림 |
| 👽 **외계인** | 의태 | 근처 동물의 스킬을 베껴 쓰는 변칙가 |

---

### 🐶 강아지 — 우다다 (Zoomies)
**게임 마스코트. 이유 없이 폭주하는 산만함의 화신.**

- **스킬**: 갑자기 통제 불능으로 폭주하다가 가끔 옆 코스로 새요.
- **속도**: ⭐⭐⭐⭐⭐ (최고) / **파워**: ⭐ (약함)
- **대사**: "우다다다다!!!"

> 💡 한 방이 강력해요. 운만 따라주면 누구보다 빨리 치고 나갑니다.

---

### 🐱 고양이 — 캣워크 (Catwalk)
**방해 따윈 '냐옹' 하고 피해버리는 얄미운 추월자.**

- **스킬**: 잠깐 사뿐해져서 바나나·포효 같은 방해를 **확률적으로 피하고** 그 틈에 슬쩍 앞질러요.
- **속도**: ⭐⭐⭐⭐ / **파워**: ⭐⭐
- **대사**: "캣워크~ 😼" / "냐옹, 안 맞지롱"

> 💡 공격력은 약해도 방해에 잘 안 당해요. 막판까지 살아남는 끈질긴 타입.

---

### 🐒 원숭이 — 바나나 (Banana)
**장난기 폭발 트러블메이커.**

- **스킬**: 앞서가는 동물 하나를 콕 집어 바나나로 미끄러뜨려 잠깐 멈춰 세워요. 가끔 헛던지기도. 🍌
- **속도**: ⭐⭐⭐ / **파워**: ⭐⭐⭐
- **대사**: "받아라! (까득)" / "어… 빗나갔네"

> 💡 선두를 콕 집어 견제하는 맛. 1 등을 끌어내리는 데 최고예요.

---

### 🐻 곰 — 포효 (Roar)
**묵직한 산의 왕.**

- **스킬**: "크아앙!" 한 방에 **주변 동물들을 다 같이** 잠깐 움찔하게 만들어요. 광역기.
- **속도**: ⭐⭐ / **파워**: ⭐⭐⭐⭐⭐ (탱크)
- **대사**: "크아앙!!" / "으르렁!"

> 💡 무리에 둘러싸였을 때 진가 발휘. 한 번에 여러 명을 멈춰 세웁니다.

---

### 🐧 펭귄 — 빙판 (Icefield)
**앞에 빙판을 까는 얄미운 전략가.**

- **스킬**: 자기 앞쪽 길에 **빙판을 쫙** 깔아요. 다른 동물들은 미끄러져 느려지지만, **펭귄은 오히려 쌩쌩** 미끄러져요. ⛸️
- **속도**: ⭐⭐ / **파워**: ⭐⭐⭐⭐
- **대사**: "직진은 자신 있어!"

> 💡 길목을 막는 지역 견제형. 뒤따라오는 추격자를 빙판으로 늪에 빠뜨려요.

---

### 🦔 고슴도치 — 가시 (Bristle)
**붙지 마, 진짜. 가시로 톡 밀어내는 까칠 방어가.**

- **스킬**: 바로 뒤에서 바짝 따라붙는 추격자가 있으면 **등 가시를 확 세워** 뒤로 톡 밀어내고 잠깐 느리게 만들어요. 반동으로 본인은 살짝 앞으로. 🦔
- **속도**: ⭐⭐ / **파워**: ⭐⭐⭐⭐
- **대사**: "따끔! 붙지 마!" / "가까이 오지 말랬지?"

> 💡 뒤에 바짝 붙는 추격자를 카운터로 견제. 정체 구간에서 끈질기게 버팁니다.

---

### 🕷️ 거미 — 거미줄 (Abduct)
**거기 서! 거미줄로 콱! 끈적이 추적자.**

- **스킬**: 앞서가는 동물 하나를 거미줄로 콱 낚아채 **자기 등 뒤로 끌어내리고**, 거미줄에 묶어 잠깐 느리게 만들어요. 🕸️
- **속도**: ⭐⭐⭐ / **파워**: ⭐⭐⭐⭐
- **대사**: "거미줄, 걸렸다!" / "내 뒤에 붙어있어~"

> 💡 선두를 콕 집어 자기 뒤로 끌어내리는 위치 강등형. 1 등을 통째로 주저앉힙니다.

---

### 👽 외계인 — 의태 (Mimic)
**스캔 완료… 카피한다! 옆 동물 스킬을 베껴 쓰는 변칙가.**

- **스킬**: 근처에서 가장 가까운 동물을 스캔해 **그 동물의 스킬을 그대로 베껴** 발동해요. 바나나든 포효든, 누구를 스캔했느냐에 따라 매번 달라지는 와일드카드. 주변에 아무도 없으면 헛스캔 (개그). 🛸
- **속도**: ⭐⭐⭐ / **파워**: ⭐⭐⭐
- **대사**: "스캔 완료… 카피한다! 🛸" / "스캔 실패… 신호 없음"

> 💡 정해진 효과가 없는 변수 그 자체. 어떤 스킬이 튀어나올지 본인도 몰라요.

## 🎁 아이템 가이드 (확장 예정)

트랙 위에 **`?` 상자**가 둥둥 떠다녀요. 먹으면 4 가지 중 하나 랜덤 — 도박 상자! 🎲

| 아이템 | 효과 |
|---|---|
| ⚡ **번개** | 나만 빼고 다 같이 느려져! |
| 💨 **방귀 폭탄** | 뒤에 방귀 구름을 깔아 추격자를 늦춤 |
| 🐢 **유도 등껍질** | 1 등을 콱 맞춰 잠깐 스턴 (1 등이 먹으면 자폭) |
| 🌟 **별** | 무적 + 가속! 모든 방해 무시하고 질주 |

> 💡 아이템은 캐릭터 상관없이 누구나 먹을 수 있어요. 운과 타이밍이 승부를 가릅니다.

## 🏟️ 경기장 (맵)

| 맵 | 설명 |
|---|---|
| 🌿 **Grassland** | 초록 풀밭 — 기본 맵 |
| 🏜️ **Desert** | 황량한 사막 |
| 🏖️ **Beach** | 해변 모래사장 |
| 🌃 **Citynight** | 야간의 도시 |
| ❄️ **Snow** | 눈 덮인 설원 |
| 🌴 **Jungle** | 울창한 정글 |

## 🛠️ 개발자 정보

### 기술 스택
- **프레임워크**: TypeScript + Vite
- **렌더링**: PixiJS v8
- **테스트**: Vitest (단위테스트) + Playwright (E2E)

### 프로젝트 구조
```
src/
  engine/      순수 시뮬레이션 (PRNG, RaceEngine, 스킬 핸들러, 추월 로직)
  data/        캐릭터·맵·모드 데이터
  renderer/    PixiJS 렌더러 (트랙, 캐릭터, 이펙트)
  shell/       DOM UI (입력, 설정, 결과)
  transport/   통신 추상화 (v1: LocalTransport)
```

### 로컬 실행
```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # 프로덕션 빌드
npm run test       # 단위 테스트
npm run e2e        # E2E 테스트
```
