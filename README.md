# 우다다 (Woodada)

귀여운 동물들이 원형 트랙을 달려 순위를 정하는 **동물 경주 추첨 게임**.

🎮 **바로 플레이하기**: [https://kim-taehan.github.io/woodada/](https://kim-taehan.github.io/woodada/)

---

## 실행 방법

```bash
npm install
npm run dev        # http://localhost:5173 에서 실행
```

## 검증

```bash
npm run typecheck  # 타입 체크
npm run test       # 단위 테스트
npm run e2e        # E2E 테스트
npm run build      # 프로덕션 빌드
```

## 기술 스택

- **프레임워크**: TypeScript + Vite
- **렌더링**: PixiJS v8
- **테스트**: Vitest + Playwright
